== Description ==

### Wordpress Auto scroll for reading plugin

WordPress AutoScroll plugin is developed to add an automatic scroll effect to your page for your readers. It’s a very user-friendly plugin with a flexible interface. AutoScroll is convenient to use when you want the readers to read your posts without clicking on the keyword marks or mouse. Now, you don’t have to go back and forth and click again. All you need is just click the play button and continue reading.

AutoScroll is the best plugin in WordPress that provides the key functionalities which are as follows.

= KEY FUNCTIONALITIES =

* You can set a specific scroll speed
* Works smoothly with all the major browsers including Chrome, Safari, Firefox, IE, etc.
* It allows to pause and resume the scrolling anytime
* It’s applicable to all posts categories and posts types
* Infinite scrolling
* Simple to use

= How to Use =

AutoScroll is extremely **easy to use**. You just have to take the following steps:

1. Go to the *Plugin* installation section
2. Find and Install the *AutoScroll* plugin
3. *Activate* it and enjoy your convenience

No further settings regulations are required! Right after activating the AutoScroll plugin buttons appear on your page.

Install the WordPress AutoScroll plugin and give your readers a chance to combine comfort and reading!

== Installation ==

There are two ways to install *Auto scroll for reading*: the easy way, when you install *Auto scroll for reading* from your WordPress dashboard, and the not so easy way, when you install it from WordPress.org.

* 1.1 The easiest way to enjoy Auto scroll for reading:
* 1.1.1 Login to your WordPress dashboard
* 1.1.2 Go to Plugins
* 1.1.3 Add New
* 1.1.4 Search for Auto scroll for reading
* 1.1.5 Click to install
* 1.2 The second way:
* 1.2.1 Download the zip file from https://wordpress.org/plugins/auto-scroll-for-reading/
* 1.2.2 Go to Plugins
* 1.2.3 Add New
* 1.2.4 Upload plugin
* 1.2.5 Choose file auto-scroll-for-reading.zip
* 1.2.6 Click to install

* 1.3 In order to install the *auto-scroll-for-reading* from Wordpress.org you must unzip the archive and copy the auto-scroll-for-reading folder into your plugins folder (\wp-content\plugins).
After one ofthis steps you can activate the plugin from the Plugins menu.
Once activated configure any options as desired and you can enjoy our auto-scroll-for-reading.
